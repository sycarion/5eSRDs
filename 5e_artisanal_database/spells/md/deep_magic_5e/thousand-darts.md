# Thousand Darts

- **Level**: 3 evocation
- **Classes**: Sorcerer, Warlock, Wizard
- **Casting Time**: 1 action
- **Range**: Self (120-foot line)
- **Components**: V, S, M (a set of mithral darts worth 25 gp)
- **Duration**: Instantaneous
- **Source**: Deep Magic 5e
